## 🚀 Featured Project

### 🧱 Eco-Brick Market Intelligence — End-to-End Data Analytics

> Analysed **355 brick manufacturers** across **31 Telangana districts** to identify optimal factory locations and predict closure risk for an eco-brick venture.

| Phase | Work Done |
|---|---|
| 🧹 Data Cleaning | Python + Pandas — 361 → 355 rows, 4 quality bugs fixed, assertion-validated |
| 📊 KPI Analysis | 5 derived columns, district-level summaries, feature engineering |
| 📈 Power BI Dashboard | 3-page report: Market Overview · Eco Transition · Opportunity Map |
| 📍 Location Scoring | Weighted model — Ranga Reddy ranked #1 across all sensitivity scenarios |
| 🤖 Machine Learning | Random Forest closure risk model — company size = #1 predictor |
| 💡 Business Insights | 7 written insights: fact → interpretation → recommendation |

**Stack:** `Python` · `Pandas` · `scikit-learn` · `matplotlib` · `Power BI` · `Jupyter` · `Git`
